# 📋 高機能アンケート管理システム (Survey Manager)

Python (Flask) と SQLite を使用した、Webアンケート管理システムです。
基本的な CRUD 操作に加え、リレーションシップを活用した高度なデータ連動と、モダンな Web UI を実装しています。

## 🌟 主な機能と実装内容

本プロジェクトは、以下の要件に基づき設計・開発されています。

### 1. ユーザー管理 (User)
- **新規ユーザー登録**: 名前とメールアドレスによるアカウント作成が可能です。
- **管理者自動生成**: システム初回起動時にデモ用の管理ユーザーが自動的に作成されます。

### 2. アンケート管理 (Survey / Question / Choice)
- **アンケート作成**: タイトルを設定し、新規アンケートを作成できます。
- **動的な質問追加**: 1つのアンケートに対し、複数の質問とそれぞれの選択肢を自由に追加可能です。
- **一括削除機能**: アンケートを削除すると、紐付いている質問、選択肢、および回答データがデータベースから自動的に連動して削除されます（Cascade Delete）。

### 3. 回答と集計 (Answer)
- **回答機能**: 公開中のアンケートに対して、ユーザーが回答を送信できます。
- **視覚的な集計結果**: 各質問の回答状況を、Bootstrap 5 を活用したアニメーション付きプログレスバーでリアルタイムに可視化します。

### 4. モダンな UI/UX
- **レスポンシブデザイン**: Bootstrap 5 を採用。
- **リッチな視覚効果**: グラデーションヘッダー、カードデザイン、ホバーアニメーションなどを導入し、直感的な操作感を実現しました。

## 🛠 使用技術

- **Language**: Python 3.9+
- **Framework**: Flask (Web)
- **ORM / DB**: Flask-SQLAlchemy (SQLite)
- **Frontend**: HTML5, CSS3, Bootstrap 5, Google Fonts (Noto Sans JP)

## 📂 データベース構成 (5テーブル)

SQLAlchemy を使用し、以下の 5 つのエンティティ間でリレーションシップを定義しています。
1. `User`: ユーザー情報
2. `Survey`: アンケートの基本情報
3. `Question`: アンケートに紐付く質問
4. `Choice`: 質問に紐付く選択肢
5. `Answer`: ユーザーによる回答記録

## 🚀 セットアップと実行方法

### 1. 依存ライブラリのインストール
```bash
pip3 install Flask Flask-SQLAlchemy
```
### 2. アプリケーションの起動
```bash
python3 app.py
```
### 3. アクセス
ブラウザで以下のURLを開いてください。 http://127.0.0.1:8000
